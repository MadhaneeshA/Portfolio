Project information

  • Project Name: Line follower
  • Project date: 19 October 2022

During the IIT Bombay event held at Global Academy of Technology in Bengaluru, Karnataka, a project named 'Line Follower' competed in the Robot category. The project showcased an innovative line-following robot that demonstrated exceptional capabilities in navigating predefined paths. The event took place on 19th October 2022, and the project's performance and technical prowess earned it an impressive 5th place among the participating teams. This achievement highlights the dedication, skills, and hard work put forth by the team behind the Line Follower project.
